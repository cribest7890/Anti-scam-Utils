# Anti-scam-Utils
this is the repository of the "Anti-scam Utils" browser extension

# How to install?
download the .crx file from the files or from the release page
then go on your favourite chromium-based browser, visit: chrome://extensions and turn on developer options, then drag the .crx file on the page and accept the popup that asks if you want to add the extension, and you're done!!
